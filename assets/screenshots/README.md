# Screenshots

This folder contains screenshots of QuickReq for documentation purposes.

## Files needed:
- `main-ui.png` - Main interface screenshot showing the request form, response panel, and key features

## Screenshot Guidelines:
- Use a clean, professional browser window
- Show the app with some sample data (not empty)
- Capture at 1200x800 or similar resolution
- Use PNG format for crisp UI elements
- Consider showing both light theme and key features

## Usage:
Screenshots are referenced in the main README.md file to give users a quick preview of the application interface.
